C MySQL API Learning
====================

Learning C through example

Pre-Reqs
========

* UNIX
* gcc (https://gcc.gnu.org/)
* libmysqlclient (https://dev.mysql.com/doc/refman/5.0/en/c-api.html)
* make (http://linux.die.net/man/1/make)

How-To
======

Create database

.. code:: bash

    $ make create_db
    $ ./create_db
